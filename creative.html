About This Project

“Infinite Camping World” is an interactive procedural map built with p5.js for a generative design course. It generates an endlessly scrolling isometric landscape using seeded Perlin noise, allowing each world seed to produce a varied environment.

The terrain is composed of water, grass, and dirt tiles, with decorations such as trees, flowers, and grass placed based on simple rules. The camera can be moved using arrow keys or WASD, and players can place tents on valid land tiles to interact with the world.

This project is part of a three-part interactive series focused on environmental themes, including responsible camping, deforestation, and reforestation. My goal was to explore how procedural systems can communicate ideas about land use by constraining player actions within the generated space.

Through this project, I worked with noise-based generation, rule-based placement, and camera navigation.
